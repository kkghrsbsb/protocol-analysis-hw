# protocol-analysis-hw

《互联网协议分析》课程作业仓库。

每次作业是一个独立的子目录（`hw1/`、`hw2/` …），统一使用 **Node.js + Vite** 构建，`npm run build:html` 产出可直接双击打开的单文件 `dist/index.html`。

## 快速开始

```bash
cd hw1
npm install
npm run dev          # 开发预览
npm run build:html   # 构建单文件产物
```

项目详细说明、作业列表、构建规范见 [`docs/src/README.md`](docs/src/README.md)。
